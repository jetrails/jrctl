## jrctl database delete

Delete a database

```
jrctl database delete DB_NAME [flags]
```

### Options

```
  -h, --help              help for delete
  -q, --quiet             only display confirmation code
  -t, --tag stringArray   filter nodes using tags (default [localhost])
```
