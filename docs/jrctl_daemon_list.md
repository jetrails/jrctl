## jrctl daemon list

List configured daemons

### Synopsis

List configured daemons.

```
jrctl daemon list [flags]
```

### Examples

```
jrctl daemon list
jrctl daemon list -t web
```

### Options

```
  -t, --tag string   specify daemon tag selector
  -h, --help         help for list
```
