## jrctl daemon list

List configured daemons

### Synopsis

List configured daemons.

```
jrctl daemon list [flags]
```

### Examples

```
jrctl daemon list
jrctl daemon list -t www
```

### Options

```
  -h, --help          help for list
  -t, --type string   specify daemon type selector
```
