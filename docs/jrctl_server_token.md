## jrctl server token

Displays token information for all configured servers

### Synopsis

Displays token information for all configured servers.

```
jrctl server token [flags]
```

### Examples

```
jrctl server token
```

### Options

```
  -h, --help          help for token
  -q, --quiet         output as little information as possible
  -t, --type string   specify server type selector
```
