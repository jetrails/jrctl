## jrctl website list

Display configured websites

```
jrctl website list [flags]
```

### Examples

```
jrctl website list
jrctl website list -q
```

### Options

```
  -h, --help              help for list
  -q, --quiet             only display site names
  -t, --tag stringArray   filter nodes using tags (default [default])
```
