## jrctl database link

Add database user to specific database

```
jrctl database link USER@FROM DB_NAME [flags]
```

### Options

```
  -h, --help               help for link
  -q, --quiet              display no output
  -t, --type stringArray   filter servers using type selectors (default [localhost])
```
