## jrctl server ingest

Ingest server token and save it to config

### Synopsis

Ingest server token and save it to config.

```
jrctl server ingest [flags]
```

### Examples

```
  echo -n TOKEN | jrctl server ingest -t localhost
```

### Options

```
  -h, --help          help for ingest
  -t, --type string   specify server type selector
```
